# Racoon Rush (Videojuego Android)

Este proyecto es un videojuego 2D sencillo desarrollado como parte de la **Actividad 1: "Crea tu primer mini-juego 'Evita los obstáculos'"** de la asignatura de **Programación Multimedia y Dispositivos Móviles**.

El juego consiste en controlar un personaje que se desplaza horizontalmente para esquivar obstáculos que caen desde la parte superior de la pantalla. La generación de obstáculos es aleatoria y la dificultad aumenta progresivamente.

El objetivo principal es poner en práctica la lógica de videojuegos en Android, el uso de _sprites_, la detección de colisiones, la gestión de audio (música y efectos) y el flujo de navegación entre menús.

## Características Principales

* **Control Táctil:** Movimiento horizontal del personaje.
* **Dificultad Progresiva:** Los obstáculos aumentan su velocidad con el tiempo.
* **Audio:** Música de fondo en bucle y efectos de sonido al colisionar.
* **Interfaz:** Menú inicial (Jugar/Salir) y pantalla de Game Over con puntuación final.

## Cómo Ejecutar el Proyecto (PC)

1.  Clona este repositorio en tu máquina local.
2.  Abre el proyecto con Android Studio.
3.  Ejecuta la aplicación en un emulador de Android o en un dispositivo físico.

## Cómo Ejecutar el Proyecto (Android)

1.  Descarga la APK de la _release_.
2.  Instala la aplicación.
3.  Ejecuta la aplicación normalmente.
